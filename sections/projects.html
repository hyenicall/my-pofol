GAIA Meeting

period 2023.12 ~

GAIA 업무 협업 툴에 통합된 인앱 미팅 서비스인 GAIA Meeting은 다양한 디바이스와 네트워크 환경에서도 안정적으로 작동하는 화상 회의 기능을 제공하여 사용자 경험과 생산성을 향상시키는 것을 목표로 합니다.
𐤏 업무 협업 툴 내에서 손쉽게 접근 가능한 화상 회의 기능 제공합니다
𐤏 다양한 디바이스(PC, 태블릿, 스마트폰)와 네트워크 환경에서 원활하게 작동하는 반응형 웹 서비스로 구현되었습니다
𐤏 효율적인 스트림 관리 및 사용자 친화적인 인터페이스 제공합니다

프로젝트 참여기능 명세

1. 서버 요청/응답 인터페이스 설계

  • ‧ 서버와 클라이언트 간의 DTO인터페이스 설계를 진행 함.
  • ‧ 미팅 서버의 데이터베이스 테이블을 고려해 데이터 교환의 최적화를 고려하여 설계를 진행 함
  • ‧ Git을 이용하여 타입의 상태관리를 진행하여 서버와 클라이언트에서 최신 수정사항을 바로 반영할 수 있도록 관리 함

2. 다양한 뷰 전환

메인 뷰 호스트 또는 주화자의 화면을 메인으로 고정하여 표시
포커스 뷰 메인 뷰와 함께 다른 참여자들의 화면을 서브뷰로 세로스크롤 형태로 표시
그리드 뷰 참여자들의 영상을 바둑판 형식으로 페이지로 표시

  • ‧ 다수의 참여자가 동시에 접속해 있을 때, 클라이언트의 CPU 및 메모리 자원도 불필요하게 소비하지 않도록 리소스를 관리함

3. 문서공유/영상공유 기능

  • ‧ 호스트/참여자의 문서/영상을 공유 할 수 있도록 하는 기능
  • ‧ 공유된 문서와, 영상의 같은 시점을 보며 미팅을 진행할 수 있도록 기능 구현
  • ‧ 단순히 화면을 공유하는 것이 아니라 미디어 서버로 부터 파일의 이미지를 다운 받아 같은 페이지를 보도록 하여 우수한 화질로 미팅을 진행하도록 구현 함
  • ‧ 공유 된 자료들을 각 참여자가 파일로 다운 받을 수 있도록 구현

4. 녹화

  • ‧ 미팅 진행 내용을 녹화하여 파일로 저장하는 기능
  • ‧ 미팅 진행과정의 화면과 오디오를 녹화하여 저장 후 미팅 종료 이후에 재생, 재확인 가능하도록 하는 기능
  • ‧ 미팅화면의 내용과 오디오를 함께 미디어 서버에 저장

5. 미팅 일정관리(예약, 반복예약, 팀원 초대)

  • ‧ 업무 협업 인앱서비스 목적에 맞게 캘린더를 통해 미팅 일정을 관리하도록 기능을 구현 함
  • ‧ 미팅 일정을 수정하거나 참여자 변동이 있을 경우 캘린더에서 접근 가능하도록 구현 함
  • ‧ 미팅 정책과, 캘린더 정책, 룸 정책을 고려하여 충돌없이 작동하도록 인터페이스와 정책 설계

6. 주화자감지

  • ‧ 발화자 감지를 하여 클라이언트에서 주화자가 누구인지 안내하는 기능
  • ‧ 인터페이스 설계와 해당 기능 클라이언트 담당하여 화면 구현

6. PIP화면 구현

  • ‧ 화면공유, 문서공유, 주화자 기능을 사용할 경우 공유자/발화자 영상을 pip화면으로 볼 수 있는 기능 구현
  • ‧ 미팅 내의 추가 화면을 구현하여 drag-and-drop형식으로 자유롭게 화면 내에 이동 할 수 있도록 구현 함
