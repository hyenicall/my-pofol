프로젝트 주요 기능

다양한 디바이스와 네트워크 환경에서도 원활하게 작동하는 반응형 웹 서비스 구현, 효율적인 스트림 관리와 사용자 인터페이스 최적화

2. 다양한 뷰 기능 (그리드, 포커스)

대용량 참여자에 맞도록 화면을 다양하게 구성함

3. 파우더룸

미팅 입장 전 장비점검, 화면설정 진행

4. 문서공유 / 영상공유

호스트가 제공하는 문서파일을 미디어서버를 이용해 모든 참여자가 같은 페이지를 높은 화질로 볼 수 있도록 구현함, 해당 페이지에서 강사는 화이트 보드를 이용해서 판서를 진행 할 수 있음

5. 화이트보드

강사는 수업내용을 화이트 보드를 이용해서 판서를 제공할 수 있음 강사 뿐만 아니라 권한이 주어진 참여자는 화이트 보드에 작성가능하고 모든 참여자가 같은 화면을 볼 수 있는 기능

6. 퀴즈/투표

수업 혹은 미팅 진행 중 퀴즈를 제공하거나, 투표를 진행 할 수 있음, 정답률과 투표결과를 호스트가 관리 가능함

7. 소그룹

미팅 참여자들을 다시 재 구성하여 미팅 내의 소그룹으로 미팅을 진행하도록 하는 기능, 수업(미팅) 중 작은 토론이나 회의가 필요할 경우 진행

8. 반응형 모바일

반응형 구현으로 다양한 디바이스에서도 미팅서비스를 사용가능하도록 구현함 웹, 모바일 모두 동일기능 사용가능

나의 담당 기능

  • 서버 요청/응답 인터페이스 설계
  • 뷰 모드 구현(메인뷰, 그리드뷰(MCU,SFU), 포커스뷰)
  • 문서공유, 영상공유
  • 주화자 감지
  • 녹화기능
